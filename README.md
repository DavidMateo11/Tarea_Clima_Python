# Sistema de Gestión de Inventarios Mejorado

Este proyecto permite gestionar productos mediante una interfaz de consola, asegurando que la información no se pierda al cerrar el programa.

## Funcionalidades
- **Almacenamiento:** Uso de archivos `.txt` para persistencia de datos.
- **Manejo de Errores:** Control de excepciones para archivos inexistentes o sin permisos.
- **Modularidad:** Código organizado en clases y archivos separados (`main.py`, `inventario.py`, `producto.py`).
